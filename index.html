    Estos son algunos proyectos de java públicos y de pago, usando la api bungee y spigot creados para distintos servidores y usos personales por diversión.



    Lobby System

    LobbySystem es un plugin de la API de bungee con el objetivo de crear y administrar el sistema de lobbys de tu network.

    Lobby Addon

    Spigot Addon de LobbySystem, permite usar la API para transportar a los jugadores a otras lobbys vía bungee.

    IPAuthenticator
