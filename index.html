Lobby System


Lobby System es un plugin BungeeCord que te permite crear y administrar las lobbys de tu network. Con este plugin puedes crear, remover, listar las lobbys del servidor, puedes transportar a uno o más jugadores a una lobby a otra, cuentas con una blacklist de servidores en los cuales no se podrá validar la ejecución del comando lobby


para más información puedes visitar la pagina de spigot o el repositorio de github.
